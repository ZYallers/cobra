# cobra
Go language command tool based on [cobra](github.com/spf13/cobra)

```bash
~: go run main.go -h

Usage:
   [command]

Available Commands:
  delKey      优雅删除Redis指定前缀的keys
  help        Help about any command
  word        单词格式转换

Flags:
  -h, --help   help for this command

Use " [command] --help" for more information about a command.
```
